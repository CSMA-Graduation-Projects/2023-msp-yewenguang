# 环境配置说明模板

本文件用于记录论文方法、实验和系统运行所需环境。学生补充时应尽量写明精确版本，避免只写“最新版”。

## 一、Python 版本

- 推荐版本：待填写，例如 Python 3.10 或 Python 3.11。
- 实际实验版本：待填写。
- 版本确认命令：

```bash
python --version
```

## 二、依赖管理

请说明本项目使用的依赖管理方式：

- Python 依赖：待填写，例如 `requirements.txt`、`environment.yml`、`pyproject.toml`。
- 前端依赖：待填写，例如 `package.json`、`pnpm-lock.yaml`。
- 系统依赖：待填写，例如编译器、测试框架、Docker、数据库。

建议提交以下文件：

- `requirements.txt` 或 `environment.yml`
- `config.example.yaml`
- `package.json`
- `Dockerfile` 或部署脚本

## 三、模型接口

本项目涉及多模型代码生成、多智能体分析和自动修复，需记录模型接口信息：

| 配置项 | 说明 | 是否可提交 |
|---|---|---|
| 模型名称 | 使用的模型或服务名称 | 可提交 |
| 接口地址 | 公共接口地址或服务类型 | 可提交，私有地址需脱敏 |
| API Key | 模型访问密钥 | 不可提交 |
| 温度和采样参数 | 实验参数 | 可提交 |
| 最大 token | 输入输出长度限制 | 可提交 |
| 调用时间 | 实验运行日期 | 可提交 |

请使用环境变量或本地配置管理密钥，并提交 `config.example.yaml` 作为样例。
